md （开发中）
===========

__md__ 是魔豆路由器的一个命令行客户端，使用 Go 语言进行开发，快速、简单、跨平台。   
使用了魔豆路由器 [API](https://github.com/modouwifi/modouwifi-api)。


### 使用

```
$ md <command> [options] [arguments]
$ md help
$ md help <command>
$ md system
```


### 命令


* `login`
* `system`
* `security`
* `wan`
* `wifi`
* `lan`
* ...
