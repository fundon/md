md (WIP)
========

A Fast Modou CLI client.   
Built with [Modou HTTP API](https://github.com/modouwifi/modouwifi-api).


## Installation

```
$ go install thub.com/modouwifi/md
```


## Usage

```
$ md <command> [options] [arguments]
$ md help
$ md help <command>
$ md system
```

### Commands

* `login`
* `system`
* `security`
